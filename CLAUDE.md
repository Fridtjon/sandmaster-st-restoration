# Claude Code Context

This repository documents the restoration of a 1994 Sandmaster ST Nitro Truck RC car.

## Project Context

- Owner has no prior experience with RC cars or nitro engines
- The truck was purchased at a flea market in unknown condition
- This is a learning project as much as a restoration project

## Documentation Guidelines

- All documentation should be written in English
- Photos will be added to document the truck's condition
- Research findings about the truck, parts, and nitro RC basics should be documented

## Key Questions to Resolve

- What manufacturer made the Sandmaster ST?
- What engine/motor does it use?
- What's needed to start and run a nitro RC car?
- What is the current condition of this specific truck?
- What parts need to be replaced or repaired?
