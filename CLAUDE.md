# Claude Code Context

This repository documents the restoration of a 1994 Kyosho Sandmaster ST Nitro Truck RC car.

## Project Context

- Owner has no prior experience with RC cars or nitro engines
- The truck was purchased at a flea market in unknown condition
- This is a learning project as much as a restoration project

## Documentation Guidelines

- All documentation should be written in English
- Photos are organized in `images/` with descriptive names
- Manual pages are in `docs/manual/` organized by topic
- Research findings should be documented as we go

## Known Information

- **Manufacturer**: Kyosho
- **Model**: Sandmaster ST (1:10 scale stadium truck)
- **Engine**: Kyosho GS11X nitro engine
- **Radio**: Hitec Ranger II 2-channel AM system (27 MHz)
- **Servo**: Hitec HS-300

## Key Questions Still to Resolve

- What's needed to start and run a nitro RC car? (fuel, glow starter, etc.)
- What is the internal condition of this specific truck?
- Does the engine turn over? Is it seized?
- Are the fuel lines and tank in good condition?
- Does the radio system still work?
- What parts need to be replaced or repaired?

## File Organization

- `images/truck/` - Photos of the truck's current condition
- `images/transmitter/` - Radio transmitter photos
- `images/accessories/` - Charger and other parts
- `docs/manual/assembly/` - Assembly instruction pages
- `docs/manual/radio/` - Radio system documentation
- `docs/manual/other/` - Parts lists, engine docs, troubleshooting
